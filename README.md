# Paris 2026 — Trip Itinerary

An interactive, day-by-day map and itinerary for a September 2026 trip to Paris.

## 🔗 Live site

**https://alexandra677.github.io/paris-site/**

## About

A single self-contained `index.html` — an interactive Leaflet map paired with
a day-by-day itinerary. Toggle any day from the legend to show or hide its
route on the map and its cards in the itinerary column.

- **Sept 5–12, 2026** · seven nights, three hotels
- Each stop has times, hours, notes, and a photo
- No build step — just open `index.html`

## Local preview

Open the file directly, or serve it locally:

```sh
python3 -m http.server 4317
# then visit http://localhost:4317
```

## Deployment

Hosted on GitHub Pages from the `main` branch. Push to `main` and the site
redeploys automatically at the link above.
